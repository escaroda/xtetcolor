# xtetcolor
Tetcolor game clone (X Window)

Original idea - Sergey Sotnikov, 1991

Tested on:
  - Raspbian/RPi3
  - FreeBSD11/amd64
  - Fedora13/amd64

Controls:
  * ENTER       - start/reset game
  * Left/Right  - move figure
  * Up/Down     - rotate figure
  * Space       - drop figure down
  * P           - pause/resume game
  * ESC         - leave game
